## Available Scripts

### `yarn start`

Runs the app in the development mode

### `yarn test`

Run Eslint tests

### `yarn testcafe`

Run Testcafe tests

### `yarn build`

Builds the app for production to the `build` folder

### `yarn sync`

Synchronize translation with `en.json`
